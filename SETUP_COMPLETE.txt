🎓 HOW TO USE:

1. IMPORT AND USE AN ALGORITHM:
   ──────────────────────────────
   from supervised.regression.linear_regression import LinearRegression
   from utils.data_preprocessing import split_data
   
   model = LinearRegression(learning_rate=0.01)
   model.fit(X_train, y_train)
   predictions = model.predict(X_test)
   score = model.score(X_test, y_test)

2. ADD A NEW ALGORITHM:
   ─────────────────────
   a) Copy TEMPLATE.py to the correct folder
   b) Implement the algorithm
   c) Update __init__.py
   d) Add example usage
   e) Run and test

3. EVALUATE YOUR MODEL:
   ────────────────────
   from utils.metrics import accuracy, f1_score, mse, r_squared
   
   acc = accuracy(y_true, y_pred)
   f1 = f1_score(y_true, y_pred)
   error = mse(y_true, y_pred)

💡 BEST PRACTICES FOR YOUR PROJECT:

✅ DO:
   • Always normalize/standardize data before training
   • Split data into train/test BEFORE any preprocessing
   • Use meaningful variable names (X_train, y_train, etc.)
   • Write docstrings for all functions and classes
   • Include example usage in if __name__ == "__main__"
   • Set random_state for reproducibility
   • Validate input shapes and types
   • Comment on complex logic

❌ DON'T:
   • Don't train and test on same data (data leakage)
   • Don't forget to normalize data
   • Don't use ambiguous variable names (x, data, etc.)
   • Don't skip docstrings
   • Don't forget random_state
   • Don't assume shapes are always correct
   • Don't leave code without comments
